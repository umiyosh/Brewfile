# My Brewfile

# a precondition

* install xcode from AppStore. And Command line tools.
* <code>ssh-keygen -t rsa</code> and add setting to Deploy key

# bootstrap

* Brewfile install

~~~ bash
% git clone https://github.com/umiyosh/Brewfile.git
% cd Brewfile
% ./bin/brewSetup.sh
~~~
